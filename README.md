# K-Downloader
<p>K-Downloader는 동영상 다운로드 툴입니다.
<p>yt-dlp에 기반하여 YouTube 등 다양한 스트리밍 서비스의 동영상을 다운로드할 수 있습니다.
<br>사용자 편의를 고려하여 GUI 환경을 제공합니다.
